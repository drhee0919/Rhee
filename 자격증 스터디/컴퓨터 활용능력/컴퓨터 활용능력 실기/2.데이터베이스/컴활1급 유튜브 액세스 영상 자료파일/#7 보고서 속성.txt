<07_보고서 속성 정리>
 보고서 관리를 위한 다양한 보고서 속성 지정을 알아본다.  

- 보고서의 보기 방식 
테이블과 폼 처럼 보고서에도 여러 보기 방식이 존재한다. 
보고서보기(거의 안쓴다), 디자인보기, 그리고 보고서 미리보기 

문제1. 일일대여현황 보고서를 완성하시오. 
1) 대여목록 쿼리를 레코드 원본으로 설정하시오. 
2) 대여일자 필드를 기준으로 내림차순으로 정렬하되 동일한 대여일자에서는
전화번호 필드를 기준으로 오름차순으로 정렬되어 표시하도록 설정하시오.
(※ 보고서에서의 정렬과 그룹화는 무조건 시험에 나온다!
→ 리본메뉴 [보고서 디자인 도구] - [디자인] - 그룹화 및 정렬 선택 
    '그룹추가', '정렬추가' 선택가능 - 정렬추가 선택
   필드([대여일자]) 를 선택하고 '내림차순' 선택 
→ '정렬추가' 로 기준을 하나 더 추가한 뒤 - 필드([전화번호]) 선택 후 '오름차순' 선택 
    (2차 정렬 완성))

 
3) 대여일자 필드에 대하여 그룹 바닥글 영역을 만들고 보고서 바닥글의 모든
컨트롤들을 대여일자 바닥글 영역으로 옮기시오. 
(※ 그룹화를 하겠다는 소리 
→ 리본메뉴 [보고서 디자인 도구] - [디자인] - 그룹화 및 정렬 선택 
    (이미 만들어 놓은 정렬이 보임 - '자세히' 버튼 클릭)
→ '바닥글 표시 안함' 을 표시로 바꿔줌 (대여일자 바닥글 생성확인) 
)
4) txt누적금액 컨트롤에는 대여일자 그룹별 금액 필드의 누계 값이 표시되도록 
하시오.
(→ txt누적금액 컨트롤 선택 - 모두 - 컨트롤 원본 - 금액 선택 
			       - 누적 합계 - 그룹 선택 
 
5) 대여일자 바닥글의 txt대여건수는 해당 대여일자에 대여한 건수가 표시되고, 
txt소계는 해당 대여일자에 대여한 금액의 합계가 표시하시오. 
(→ txt대여건수 선택 - 모두 - 컨트롤 원본 - '=Count(*)' 입력 
 ※asterisk는 '모든 레코드' 를 지칭하지만 대여일자 바닥글이기 때문에 '그룹내 모든 레코드'가 지정된다.
   따로 그룹설정 할 필요 X 

6) 대여일자 필드의 값이 이전 레코드와 동일한 경우에는 표시되지 않도록 설정하시오.
(→ 대여일자 텍스트 상자 컨트롤 선택 - 모두 - 중복내용 숨기기 - '예' 선택 )
 
7) 대여일자가 바뀌면 새 페이지에 표시되도록 대여일자 바닥글을 설정하시오. 
(→ 대여일자 바닥글 선택 - 모두 - 페이지 바꿈 - 구역 후 선택 )

8) 페이지 바닥글의 txt페이지에는 페이지를 1 / 10 형태로 표시되도록 설정하시오. 
- 1은 현재페이지고 10은 전체페이지임 
(※ '=page' : 현재 페이지를 찍는 명령어 
    '=pages': 전체 페이지를 찍는 명령어 
→ 'txt페이지' 컨트롤 선택 - 컨트롤 원본 - '=[Page] & " / " & [Pages]' 입력 
※ & 연산자 입력 시에는 연산자 양옆에 공백을 남길 수 있도록 함) 


9) 페이지 바닥글의 txt날짜에는 현재 시스템의 날짜 및 시간이 표시되도록 설정 
- 연 월 일 오전/오후 시 분 초 
(→ txt날짜 컨트롤 선택 - 모두 - 컨트롤 원본 - '=Now()' 입력 
			  - 형식 - 'yyyy년 m월 d일 ampm hh시 nn분 ss초' 입력
