<06_폼 컨트롤 속성 정리>
폼의 다양한 컨트롤 들의 속성 지정을 해보고 익혀보자 

문제1. 사원별 점수 관리 폼을 완성할 것 
1) 본문의 txt사원번호, txt성명, txt소속, txt사점, txt감점 컨트롤에 컨트롤 원본을
각각 사원번호, 성명, 소속, 사점, 감점 필드를 바운드 시키시오. 
(→ 컨트롤 원본에 각각 바운드) 
2) 본문의 txt총점, txt추가점수 컨트롤은 다음고 같이 계산할 것 
- txt총점: 가점 + 감점 (→컨트롤 원본 - '=[가점]+[감점]' 입력)
- txt추가 점수: 총점의 10%(※[총점]이란 레이블은 없다. 단, 컨트롤도 계산에 이용가능하다
			→ 컨트롤 원본에 '[txt총점] * 0.1' 입력) 

3) 폼 머리글에 사원별 점수 관리라는 레이블 컨트롤을 생성하시오. 
- 컨트롤 이름은 LBL_제목, 굴림체, 18pt, 굵게, 가운데 맞춤, 그림자 효과 사용
(→ [폼 디자인 도구] - [디자인] - 컨트롤 그룹에서 '레이블' 선택 후 폼 머리글에 생성
     글꼴은 [폼 디자인 도구] - [형식] - 글꼴에서 조정
     속성 [모두]탭 - 이름에 LBL_제목 입력
     속성 [모두]탭 - 특수효과 - 그림자 선택 )
 

4) 본문의 모든 텍스트 상자 컨트롤의 특수 효과를 오목으로 설정하고 테두리
스타일을 투명으로 설정할 것 
(→본문 컨트롤들 복수 지정후 [모두] 탭 - 특수효과 - 오목
    			          - 테두리 스타일 - 투명 선택 )

5) txt사원번호 컨트롤은 포커스를 가질 수 없도록 설정 
(→ [모두] - 사용가능 에서 아니오 선택
 이제 폼 보기 화면에서 사원번호 컨트롤을 클릭 및 편집할 수 없다.(접근금지)
 ※ 포커스를 가질 수 없다? → 일단 클릭을 해야 커서가 끔뻑끔뻑하니 원천차단 → 사용불가 설정 
)

6) 폼 바닥글 하단에 가점의 합계를 구하는 컨트롤 생성 
- 텍스트 상자 이름은 txt가점합계, 레이블 캡션은 가점합계로 설정할것 
(→ [폼 디자인 도구] - [디자인] - 컨트롤 에서 '텍스트 상자' 를 생성하면 
    레이블과 텍스트 상자가 동시에 생성된다. (*만약 텍스트 상자 마법사가 뜨면 그냥 취소버튼클릭)
    텍스트 상자 클릭 후 [모두] - 이름 - txt가점합계 입력 
			 - 컨트롤 원본 - '=sum([가점])' 입력 
    레이블 클릭 후 [모두] - 캡션 - 가점합계 입력 

7) 본문의 txt가점, txt감점, txt총점, txt추가점수, txt가점 합계 컨트롤은 데이터를 
편집할 수 없도록 설정
(※ 컨트롤들은 ctrl, shift를 이용해서 복수 선택이 가능하다. 
 ※ 5)번의 [사용가능] 보다는 조금 낮은 보안 수준인 [잠금] 설정 가능  
 →[모두] - [잠금] - 예 선택) 

8) 모든 숫자 데이터 컨트롤은 숫자로 설정하고 오른쪽 맞춤으로 정렬하세요
(※ 숫자 데이터 컨트롤들은 7) 과 동일하다.
 → [모두] - 형식 - 일반숫자 선택 
    리본 메뉴 [폼 디자인 도구] - 형식 - 글꼴에서 오른쪽 맞춤 선택 ) 

9) 본문의 모든 컨트롤들이 왼쪽부터 차례대로 정지할 수 있도록 탭 순서를 설정하세요
(→ 리본메뉴 [폼 디자인 도구] - [디자인] - [탭 순서] - 마우스로 순서 조정
 ※ 하단의 '자동순서' 버튼으로도 조정이 가능하다)

10) 폼 바닥글 왼쪽 하단에 폼을 닫을 수 있는 폼닫기 단추를 생성하시오.
- 컨트롤의 이름은 cmd닫기, 캡션은 폼닫기로 설정 
(※ '명령 버튼' 컨트롤의 경우는 마법사로 간단하게 생성가능 
 → [폼 디자인 도구] - [디자인] - 컨트롤 - 명령단추 선택 후 폼 바닥글에 생성 
     마법사 팝업시 폼-닫기 선택 후 다음, 캡션과 이름 순서대로 설정 
 ※ [폼 디자인 도구] - [정렬] - 크기 및 순서 조정 탭에서 컨트롤들의 간단한
    크기 및 위치 조정 가능 )
