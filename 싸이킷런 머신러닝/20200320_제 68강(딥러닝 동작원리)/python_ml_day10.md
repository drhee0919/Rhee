**지난과정 복습** 

> - 인공지능(AI)
>
>   인간과 비슷하게 사고하는 컴퓨터 지능을 포괄, 인간이 가진 지적 능력을 컴퓨터를 통해
>
>   구현하는 기술 
>
> - 머신러닝
>
>   인공지능의 하위분야. 컴퓨터가 스스로 학습하여 인공지능의 성능을 향상 시키는 기술
>   알고리즘을 이용해서 데이터를 분석 → 분석을 통해 학습 → 학습한 내용을 기반으로 분류, 예측, 군집 등을 수행 
>
> - 딥러닝 
>
>   인간의 뉴런과 비슷한 인공신경망 방식으로 정보를 처리 → 아주 복잡한 비선형 문제를 인공신경망 이론을 기반으로 스스로 학습, 해결할 수 있도록 하는 머신러닝, 인공지능의 하위분야 
>
>   인공신경망은 넓게 여러 층으로 구성(심층 신경망 DNN)
>
>   딥러닝으로 군집화되어 있는 복잡한 비선형 데이터를 분류 → 머신러닝 알고리즘으로 학습 → 의사 결정, 예측, 판단 수행 	
>
>   딥러닝은 입력층, 하나 이상의 은닉층, 출력층으로 구성된 계층 구조로 구성
>
>   ex/ Keras는 Theano와 Tensorflow 를 래핑한 딥러닝 라이브러리 
>
>   ```
>   Sequential() - 입력층, 은닉층, 출력층 등 계층 구조를 구성하기 위한 keras
>   의 클래스 
>   
>   Dense(출력 뉴런수, input_dim=입력 뉴런수, init=가중치 초기화 방법,  activation=활성화함수) - 각 층이 어떤 특성을 가지는지 옵션을 설절
>   딥러닝은 입력값에 가중치를 곱하고 그 모든 합이 활성화함수를 통해 임계치를 초과하면 출력값이 발생
>   
>   activation는 다음층으로 어떻게 값을 넘길지를 결정 (relu, sigmoid, softmax)
>   
>   loss 는 신경망이 한번 실행될 때마다 오차를 추적하는 함수
>   
>   optimizer는 오차를 줄이는 함수를 지정
>   
>   fit() - 학습
>   
>   evaluate()  - 정확도 예측
>   
>   ※keras로 머신러닝을 수행할때는 input 데이터는 반드시 Numpy 배열 형태로 전달해야 합니다.
>   ```
